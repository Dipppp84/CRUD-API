# CRUD-API


[](#request)Request path :
* GET      
  /api/users

* POST     
  /api/users
* GET      
  /api/users/id
* PUT      
  /api/users/id
* DELETE   
  /api/users/id

[](#examples)Example :

  `http://localhost:3000/api/users` "GET" - get all users


[](#install)Install :

  `npm install`

[](#modes)Modes :
* development 
  `npm run start:dev`
* production
  `npm run start:prod`
* load balancer
  `npm run start:multi`
